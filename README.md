# WebTrees

Aplicación web integral para la gestión administrativa y operativa. Desarrollada con un enfoque moderno, rápido y escalable.

## 🚀 Tecnologías y Stack

El proyecto está construido utilizando las siguientes tecnologías principales:

### Frontend
- **[React 19](https://react.dev/)**: Biblioteca principal para la construcción de interfaces de usuario.
- **[Vite 8](https://vitejs.dev/)**: Entorno de desarrollo ultrarrápido y empaquetador de módulos.
- **[React Router DOM v7](https://reactrouter.com/)**: Enrutamiento dinámico para la navegación entre vistas y paneles (incluyendo rutas protegidas para administradores).

### Estilos y Diseño
- **[Tailwind CSS v4](https://tailwindcss.com/)**: Framework de CSS basado en utilidades para un diseño a medida, rápido y responsivo.
- **[Framer Motion](https://www.framer.com/motion/)**: Animaciones fluidas e interacciones dinámicas en la interfaz.
- **[Lucide React](https://lucide.dev/)**: Biblioteca de iconos consistentes y ligeros.
- **[Lottie React](https://lottiefiles.com/)**: Renderizado de animaciones vectoriales ligeras.

### Backend y Base de Datos (BaaS)
- **[Supabase](https://supabase.com/)**: Backend as a Service utilizado para la autenticación de usuarios, base de datos en tiempo real (PostgreSQL) y gestión de roles. El proyecto integra múltiples bases de datos (Ej: sistema principal y sistema de Posventa integrado).

### Gráficos y Utilidades
- **[Recharts](https://recharts.org/)**: Generación de gráficos y estadísticas visuales para los paneles de resumen.
- **[jsPDF](https://parall.ax/products/jspdf) & jsPDF-AutoTable**: Utilizados para la generación y descarga de presupuestos y recibos directamente en formato PDF.
- **[XLSX](https://sheetjs.com/)**: Lectura y exportación de datos en hojas de cálculo (Excel).
